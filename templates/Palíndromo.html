Código:

# Um palíndromo é uma palavra, expressão ou frase que não muda se for lida de trás para frente, ignorando letras maiúscuals, espaços e # pontuação. Exemplo: "Subi no onibus" é um palíndromo.


palavra = input("Digite uma frase ou palavra: ").replace(" ", "").lower()
# O comando input pede ao usuário que digite uma frase ou palavra, e cria uma variável "palavra" com esse valor.
# O comando ".replace(" ","")" substitui os espaços com "vazio", e o comando ".lower()" faz todos os caracteres virarem minúsculos.
# Ou seja, agora temos uma "string" (é uma sequência de caracteres no python) sem espaços nem letras maiúsculas.

n = len(palavra)
# O comando "len()" conta o número de caracteres dentro de uma string.
# No caso, cria a variável "n" igual ao tamanho do input do usuário.

t = 0
letras_iguais = 0
# Aqui criamos algumas variáveis que vão auxiliar na comparação entre as letras.


while (t < n):
# Como vamos precisar comparar cada letra da palavra, queremos criar um loop que roda "n" vezes.
# Isso é feito usando um comando while, que roda enquanto t for menor do que n (t começa em 0).
# Para isso, o valor t aumenta a cada vez que o comando while roda.

if (palavra[t]) == (palavra[(n-1)-t]):
letras_iguais += 1
t += 1
# Basicamente, essa parte do código compara cada letra da palavra com a sua correspondente de trás para frente, começando pela primeira
# e a última, depois a segunda com a penúltima, até chegar na última letra.
# Se elas forem iguais, o valor de "letras_iguais" aumenta 1.

# A notação "palavra[t]" indica a letra na palavra que ocupa a posição "t".
# Além disso, o valor de t aumenta a cada vez que o loop roda, o que faz passar de uma letra para a próxima.
# Obs.: No python, "palavra[0]" equivale á primeira letra, então a posição da última letra é "n-1" (não é o tamanho da palvra).
# Logo, temos que "palavra[t]" e "palavra[(n-1)-t]", com t = 0, são exatamente a primeira e a última letras, e assim sucessivamente,
# o que é exatamente o que queremos!


if letras_iguais == n:
print("A palavra é um palíndromo.")
else:
print("A palavra não é um palíndromo.")
# Se o tivermos a mesma quantidade de "letras_iguais" quanto letras na palavra ("n"), o programa diz que a palavra é um palíndromo!
# Isso só vai acontecer se a todas as letras forem iguais às suas correspondentes.
# Caso contrário, o programa diz que a palavra não é um palíndromo.
